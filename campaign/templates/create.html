Create a new campaign

Create a new sequence of information to be sent to the user within the current campaign

  • Select a current campaign from the active campaigns to operate from the table Current_Campaign
  • Create at least one post for each category using the mezzanine administrator. Take the id of each post and fill the table NEWSLETTER to indicate the default sequence of each post to be send to each participant in the current campaign
  • Assign the Sequence to be run to the current campaign
  • Run the assignment of posts to each participant to create the individual newsletter to be exposed in the site or send by email
